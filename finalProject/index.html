Final Project Description for 541

Summary

This website will provide a very quick primer for using an locally sourced interpreter. This is primarily designed for US military personnel who are using interpreters in tactical scenario, but will have applicability for anyone using an interpreter.

Background

During an operation, there are occasions when units will lack the linguistic ability to communicate personally and effectively with the local populace in the area of operation. The use of local interpreters is often the best or only option. The proper use and supervision of interpreters can play a decisive role in the mission. This project

Description

The goal of this project is to create a simple training course that will provide a quick review for selecting and using a local interpreter in a tactical environment. It is not meant to make the soldier an expert, but keep him from making common mistakes.

Learning Objectives

  • Soldier will correctly identify 5 of the 8 considerations for selecting an interpreter
  • Soldier will correctly identify all 5 critical things to avoid when working with a translator
  • Given a scenario with a target and an interpreter, the soldier will place him or herself in the correct position
  • The soldier will use the first-person when communicating with the target
