Introduction

This website will provide a very quick primer for using an locally sourced interpreter. This is primarily designed for US military personnel who are using interpreters in tactical scenario, but will have applicability for anyone using an interpreter.

During an operation, there are occasions when units will lack the linguistic ability to communicate personally and effectively with the local populace in the area of operation.
